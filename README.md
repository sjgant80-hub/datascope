## What It Does

DataScope is a niche-adaptive intelligence platform. You tell it what you care about — F1, gold, Thai real estate, fitness, crypto — and it becomes your personal research desk: news feed, market data, trending topics, auto-generated YouTube scripts, multi-lingual content, alerts, widgets. All in one sovereign HTML file.

Built for Michel and anyone who needs API intel without the noise.
